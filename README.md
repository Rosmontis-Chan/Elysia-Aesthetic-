## ğŸ“ Deskripsi

**Elysia Chan** adalah website generator font berbasis web sederhana. Website ini memungkinkan pengguna mengetik teks apa saja dan secara instan mengubahnya menjadi **30+ gaya font unik** (seperti *Cursive, Gothic, Bubble, Flip*, dll) yang bisa digunakan untuk bio Instagram, WhatsApp, Twitter (X), atau TikTok.

Didesain dengan tema **"Blue Sky"** yang cerah, *glowing*, dan efek animasi awan yang menenangkan.

---

## âœ¨ Fitur Utama

* **30+ Gaya Font Unik**: Termasuk Script, Bold, Terbalik (Flip), Kotak, dan banyak lagi.
* **Desain Aesthetic**: Tema Biru Langit & Putih yang cerah, *smooth*, dan *glowing*.
* **One-Click Copy**: Salin teks hanya dengan satu klik tombol.
* **Animasi Halus**: Efek awan bergerak dan transisi tombol yang *bouncy*.
* **Responsif**: Tampilan rapi di HP maupun Laptop/PC.
* **Tanpa Iklan**: Murni untuk penggunaan aesthetic.

Project ini dibuat murni menggunakan teknologi web dasar (Native/Vanilla) tanpa framework berat, sehingga sangat ringan dan cepat.
